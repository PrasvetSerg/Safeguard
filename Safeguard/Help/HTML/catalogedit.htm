Форма редактирования записи

 Top  Previous  Next

После выбора создания новой записи или изменения текущей открывается форма редактирования записи.

 

Пример формы редактирования записи

 

Изменение записи

 

Поля ввода, в зависимости от выбранного действия и состояния записи, могут быть доступны для ввода/редактирования данных или только просмотра. Поля доступные для ввода/редактирования имеют белый фон, а поля только для чтения - серый.

По окончании внесения изменений и проверки следует выполнить сохранение нажатием кнопки "Сохранить" ("Архивировать" или "Удалить" при архивации/удалении). Если данные некорректны или заполнены не все поля операция сохранения не будет произведена, а на экране появится сообщение поясняющее причину. После устранения указанных причин следует снова выполнить попытку сохранить.

 

Пример сообщения о неверно заполненных данных

 

Поведение формы

 

При нажатии кнопки подтверждения выполняемого действия "Сохранить/Архивировать/Удалить" - происходит сохранение результата изменения данных в базу и закрытие формы редактирования.

Нажатие кнопки "Отмена" - отменяет изменения и закрывает форму редактирования.

При закрытии некоторых форм редактирования, имеющих несохранённые данные, может быть выдан запрос на подтверждение сохранения данных или отмену закрытия.
