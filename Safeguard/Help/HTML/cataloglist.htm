Форма списка записей отображает данные в виде простой таблицы и позволят выполнять некоторые действия над записями.

Отображение данных можно настроить в соответствии с решаемой в данный момент задачей (см. простая таблица).

Данные таблицы в текущем представлении можно экспортировать в  файл Excel, для этого в меню программы выбрать Отчёты -> Экспорт.

Текущее представление таблицы можно сохранить в качестве постоянного, и при повторном открытии задачи таблица уже будет иметь соответствующий вид. Для этого в меню программы выбрать Вид -> Сохранить. Для возвращения таблицы к виду по умолчанию воспользоваться Вид -> Сбросить.

 

Чтобы принудительно обновить данные на форме последними сведениями из базы следует нажать кнопку "Обновить" на панели инструментов.

 

Количество видимых записей отображается под таблицей в строке статуса.

 

Управление записями

 

Управление записями выполняется с помощью панели инструментов, размещённой над таблицей с данными.

 

Для добавления новой записи можно выполнить двойное нажатие мышки на свободном поле таблицы, если она не содержит данных, или воспользоваться кнопкой "Добавить" панели. При этом отобразится форма редактирования записей с помощью которой выполняется создание новой записи. После после успешного создания новая запись появится в таблице и фокус переместится на неё, если данная запись не сокрыта фильтром.

 

Чтобы внести изменения в запись можно выполнить двойное нажатие мышки на требуемой записи или воспользоваться кнопкой "Изменить". И выполнить изменения в отобразившейся форме редактирования записей.

 

Архивирование и удаление записей производится выбором соответствующих кнопок на панели инструментов с подтверждением выбранного действия в появляющейся форме редактирования записей.
