Поля ввода данных

 Top  Previous  Next

Поля ввода данных служат ручного заполнения их данными, на основе произвольного ввода, выбора из списка имеющихся значений или на основе информации из внешнего источника.

 

Текстовое поле

 

Простое текстовое поле, обычно служит для ввода нетепизированных данных. Может ограничиваться максимально возможной длинной вводимых данных, языком ввода, наличием только литеральных символов.

 

Пример простого текстового поля

 

Текстовое поле с маской ввода

 

Текстовое поле с заданной маской ввода позволяет вводить информацию только заданного формата. Требуемые для заполнения позиции подсвечиваются знаком нижнего подчёркивания. Если поле заполнено не полностью или введённые данные не соответствуют маске ввода перед полем ввода отображается пиктограмма белого креста на красном кружке. Маскирование вводимых данных позволяет стандартизировать ввод, улучшить отображение и восприятие данных, исключить некоторые ошибки ввода.

 

Пример текстового поля с маской

Поле со списком значений

 

Поле с заданным списком значений позволяет использовать только имеющиеся значения для заполнения поля. Обычно это недоступные для изменения пользователем значения заданные на этапе создания программы. Для выбора требуемого значения поля можно начать непосредственный ввод с клавиатуры, и при наличие так же начинающихся элементов, будет предложено завершение фразы, выделенное жёлтым цветом. Либо с помощью стрелки вниз в конце поля и пролистывания открывшегося списка выбрать требуемое значение.

 

Пример поля с выбором значения из списка

 

Поле с внутренним заполнением

 

Поле связанное со справочником, аналогично полю со списком значений, позволяет заполнять данные только значениями из имеющегося списка вариантов, но взятых из справочника, Управление которым доступно пользователю. А так же имеет большую информативность, возможность сортировки и поиска значений.

Переход к просмотру справочника осуществляется нажатием пиктограммы с тремя точками в конце поля. В результате открывается форма со списком записей справочника, в которой можно так же выбрать значение двойным нажатием кнопки мыши или скорректировать список.

Чтобы очистить значение поля следует нажать крестик в конце поля.

 

Пример поля связанного со справочником

 

Поле с внешним заполнением

 

Ввод данных в поле с внешним заполнением осуществляется нажатием пиктограммы трёх точек в конце поля и выбором двойным нажатием кнопки мыши требуемой записи из списка записей открывшейся формы. Выбранная запись после закрытия окна со списком заполнит поле ввода данных.

Чтобы очистить значение поля следует нажать крестик в конце поля.
