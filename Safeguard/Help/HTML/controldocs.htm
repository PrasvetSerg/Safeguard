Журнал расстановок содержит информацию о документах, предоставленных подрядными организациями.

 

Просмотр журнала

 

Для просмотра журнала расстановок необходимо выбрать меню "Журналы" -> "Данные расстановок" боковой панели быстрого запуска или главного меню программы.

 

Пример формы журнала

 

Отображение данных можно настроить в соответствии с решаемой в данный момент задачей (см. простая таблица). Настроенное представление таблицы можно сохранить в качестве постоянного, и при повторном открытии задачи таблица уже будет иметь соответствующий вид. Для этого в меню программы выбрать Вид -> Сохранить. Для возвращения таблицы к виду по умолчанию воспользоваться Вид -> Сбросить.

Данные таблицы в текущем представлении можно экспортировать в файл Excel, для этого в меню программы выбрать Отчёты -> Экспорт.

Чтобы принудительно обновить данные на форме последними сведениями из базы следует нажать кнопку "Обновить" на панели инструментов.

В строке статуса задачи отображается количество видимых записей с учётом применённых фильтров данных.

 

Управление записями

 

Управление записями выполняется с помощью панели инструментов, расположенной над таблицей с данными.

 

Для добавления новой записи нужно воспользоваться кнопкой "Добавить" панели инструментов. При этом отобразится форма создания нового документа, в которой требуется заполнить соответствующие поля шапки документа. После успешного создания документа отобразится форма форма редактирования документа в которой можно добавить непосредственно записи расстановки.

 

Для создания документа на основе уже имеющегося, необходимо выбрать искомый документ и воспользоваться кнопкой "Копировать" панели инструментов. Дополнительно программа запросит дату, на которую следует произвести копирование. После этого отобразится форма редактирования документа заполненная скопированными данными, которые при необходимости можно изменить.

 

Чтобы внести изменения в запись нужно воспользоваться кнопкой "Изменить". И выполнить изменения в отобразившейся форме редактирования документа.

 

Архивирование и удаление записей производится выбором соответствующих кнопок на панели инструментов с подтверждением выбранного действия в появляющейся форме редактирования записей.
