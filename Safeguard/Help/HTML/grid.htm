Простые таблицы

 Top  Previous  Next

Списки не содержащие агрегируемые данные представляются простыми таблицами, поддерживающими удобные механизмы поиска, фильтрации, группировки и изменения представления данных. Все действия выполняются с помощью манипулятора "мышь".

Управление дополнительными панелями, вспомогательными окнами и некоторыми параметрами таблицы осуществляется через меню, вызываемое нажатием правой кнопки на наименовании колонок.

 

Меню управления таблицей

 

Видимые колонки данных можно группировать, менять местами, сортировать, скрывать или отображать. С помощью панелей поиска и фильтрации можно легко ограничивать объём отображаемых данных.

 

Группировка данных

 

Для выполнения группировки, наименование группирующей колонки нужно перетащить на Панель группировки данных или в меню таблицы выбрать "Группировать по этой колонке". Для разгруппировки следует перетащить наименование колонки обратно или через меню по правой кнопке мыши на панели группировки "Разгруппировать".

Сокрытие колонок данных

 

Чтобы скрыть колонку, её наименование достаточно оттащить в сторону до появления на значка с крестиком и отпустить. Чтобы обратно отобразить колонку во вспомогательном окне "Выбор колонок" найти необходимое наименование колонки и перетащить его к остальным над таблицей в требуемое место. Аналогично простым перетаскиванием наименований выполняется изменение порядка следования отображаемых колонок.

Сортировка данных

 

Чтобы отсортировать таблицу по колонке в порядке возрастания или убывания достаточно кликнуть по наименованию колонки или воспользоваться соответствующими пунктами меню таблицы. А с нажатой кнопкой "Shift" на клавиатуре можно задать сортировку по нескольким колонкам. Колонки по которым выполнена сортировка содержат в наименовании значки треугольников, указывающих направление сортировки.

Поиск данных и фильтрация

 

Таблицы поддерживают поиск/фильтрацию по всему содержимому или по каждой колонке в отдельности. В результате отображаемые данные ограничиваются только строками содержащими искомые данные или удовлетворяющие условию фильтра. Для этого может использоваться Панель поиска по всей таблице, Строка автофильтра по колонке или Настраиваемый фильтр, который позволяет наиболее гибко ограничить выводимые данные согласно заданным требованиям.

Наименования колонок, на которые настроен фильтр помечены значком фильтрации в виде воронки. Нажатие на значок воронки вызывает дополнительное меню в зависимости от типа содержимого колонки, в котором можно задать или изменить параметры автофильтра.
