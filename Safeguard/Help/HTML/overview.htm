Пользовательский интерфейс программы

 

Программа имеет стандартный для ОС Windows мультиоконный интерфейс, оптимизированный для управления с помощью манипулятора типа "мышь". Главное меню программы и панели инструментов динамически расширяются дополнительными возможностями в зависимости от активного в данный момент модуля и выполняемой задачи.

 

Основное окно программы

 

Боковая панель быстрого запуска предоставляет пользователю простой и удобный доступ к модулям, используемым при выполнении ежедневных задач:

•Справочники - содержится статичная информация.
•Журналы - оперативно изменяемая информация.
•Отчёты - информация представленная в виде заранее подготовленных для просмотра и печати форм отчётов.

 

Основное рабочее пространство отображает активные в данный момент задачи. Переключение между ними осуществляется выбором вкладки с соответствующим задаче названием или повторным выбором задачи в боковой панели быстрого запуска.

 

В верхнем правом углу окна программы отображается информация о текущем пользователе, выполнившем вход в программу.
