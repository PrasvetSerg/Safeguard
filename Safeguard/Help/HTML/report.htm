Формы просмотра отчётов отображают данные в виде готовом к печати. Позволяя распечатать отчёт, экспортировать в файл или передавать в почтовую программу.

 

Пример формы просмотра отчёта

 

Для удобства отображения данных имеется возможность изменять масштаб отображения, располагать несколько страниц рядом по горизонтали или в колонку.

Отчёты позволяют выполнять быстрый поиск по тексту. Все действия выполняются с помощью панели инструментов или меню окна.
