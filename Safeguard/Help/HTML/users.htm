Пользователи и полномочия

 Top  Previous  Next

Управление пользователями и их полномочиями осуществляется с помощью формы справочника "Пользователи программы". Вызов осуществляется через основное меню программы Сервис -> Администрирование -> Пользователи. В верхней таблице отображаются непосредственно пользователи, а в нижней группы полномочий в которые входит выделенный в текущий момент пользователь.

 

Форма пользователи программы

 

Регистрация нового пользователя программы, изменение данных существующей пользователей выполняются посредством использования кнопок панели инструментов и последующим внесением изменений на форме редактирования данных пользователя. После внесения изменений следует подтвердить сохранение изменений нажатием кнопки "Сохранить".

 

Форма изменения данных пользователя

 

Для наделения пользователя полномочиями, его необходимо включить в соответствующую группу нажав кнопку "Добавить" и выбрав требуемую группу. Для снятия с пользователя имеющихся полномочий требуется исключить пользователя из группы, выделив её и нажав кнопку "Удалить".

 

Форма изменения данных пользователя

 

Указание возможностей группы осуществляется с помощью форм Разрешения и Список процедур.

 

Правила

 

Одновременно не может быть двух пользователей с одинаковым именем входа в систему.

Если пользователь временно не должен иметь доступ к данным программы следует отметить флажком "Заблокирован".
